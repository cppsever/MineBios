## Features
> ✅ 100% Assembly
> 
> ✅ Movement
> 
> ✅ Basic physics
> 
> ✅ Interaction
> 
> ~ Only supported bios x32 pc or vm
>
> ✅ Supported Virtual-Box and Qemu

I haven't tried with real hardware but you can try with ventoy

## Compiling

Normal:

```
make
```

Iso compiling:

```
make iso
```

But you need:

```
sudo pacman -S qemu qemu-full cdrtools make nasm
```
